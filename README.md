# ecommerce-analysis
e-commerce sales analysis mini project

📊 Analisis: Superstore Sales Dataset
🛠️ Tools: Python | Pandas | Seaborn | Matplotlib
🔍 Key Findings:
• Pasar United Kingdom mendominasi 85.1% dari total revenue.
• Transaksi banyak dilakukan pada jam 10:00 - 15:00, dengan puncak transaksi terjadi pada pukul 12:00.
• Sebanyak 14.8% revenue (£1,522.5K) berasal dari Customer ID kosong (asumsi dari Guest Checkout).
• Produk 'PAPER CRAFT, LITTLE BIRDIE' menghasilkan pendapatan kotor terbesar, namun ternyata pendapatan bersihnya £0.00 karena pengembalian.
• Produk 'REGENCY CAKESTAND 3 TIER' adalah kontributor revenue terbesar.
• Kontributor revenue terbesar justru bukan dari penjualan produk, melainkan dari layanan 'DOTCOM POSTAGE'. Begitu juga dengan pengeluaran terbesar disebabkan oleh biaya layanan 'AMAZON FEE'.
